## Installation
```
$ npm install Propel -g
```

## Usage
<dl>
  <dt>Clone a starter kit</dt>

  ```
  propel
  ```
  

  <dt>Add a new starter kit</dt>

  ```
  propel -a
  ```

  <dt>Remove a start kit</dt>

  ```
  propel -r
  ```
</dl>
